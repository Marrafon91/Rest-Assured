# Rest Assured Studies

Repositório dedicado a estudos e práticas de testes de API utilizando **Rest Assured** em Java.

## 📌 Sobre
Este projeto contém exemplos e exercícios para aprender e praticar testes automatizados de APIs REST, incluindo requisições HTTP, validações de respostas e testes automatizados.

## 🚀 Tecnologias
- Java
- Rest Assured
- JUnit
- Maven

## 📚 Conteúdos estudados
- Requisições **GET, POST, PUT e DELETE**
- Validação de **status code**
- Validação de **body e headers**
- Testes automatizados de APIs REST
- Estrutura de testes com Maven
